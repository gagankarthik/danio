# create-danio

The fastest way to start a [Danio](https://www.npmjs.com/package/danio-js) app.

```bash
npm create danio@latest my-app
cd my-app
npm install
npm run dev
```

That scaffolds a ready-to-go project:

- **Vite** dev server with hot reload, wired to compile JSX with Danio's runtime
- an example app and a `jsconfig.json` for editor autocomplete
- the Danio favicon and sensible SEO `<meta>` tags in `index.html`

You can also run it directly:

```bash
npx create-danio my-app
```

Zero dependencies — it's just Node copying a template. See the
[Danio docs](https://github.com/gagankarthik/danio-js#readme) for everything else.
